## Getting Started

### **Setup**
1. **Clone/Download** the project files
2. **Add your assets** to the `assets/` folder:
   - Profile image: `assets/images/about.png`
   - Certification images: `assets/images/`
   - Resume: `assets/documents/Resume.pdf`
   - Favicon: `assets/images/favicon.ico`

3. **Update information**:
   - GitHub links in projects section
   - Social media links in footer
   - Contact information (currently: m.hasnainreactions@gmail.com, +92 322 1374013)
   - SEO meta tags for your domain

4. **Open** `index.html` in your browser

### **Customization**

#### **Theme & Colors**
- **Primary Color**: Change `#c084fc` in `css/style.css`
- **Gradient**: Modify background gradients in CSS
- **Animations**: Adjust keyframes and transitions

#### **Content**
- **Projects**: Update project cards with your actual projects
- **Skills**: Modify skills list in HTML
- **Certifications**: Update certification images and text
- **Contact**: Update contact information

#### **SEO**
- **Meta Tags**: Update title, description, keywords
- **Open Graph**: Update social media sharing info
- **Structured Data**: Update JSON-LD schema with your details
